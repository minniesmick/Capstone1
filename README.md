## Overview

Cyberpunk GUI is a powerful tool for hybrid decision-making and sentiment analysis, designed to streamline your development process with advanced features and an intuitive interface.

**Why Cyberpunk GUI?**

This project aims to simplify complex tasks through its core features:

- **🚀 Multi-Model Comparator Module:** Easily compare different models without manual effort.
- **🎨 Modern GUI with Dark Neon Theme:** Enjoy a sleek, visually appealing interface for better productivity.
- **🔍 Sentiment Analysis for Turkish Text:** Accurately classify sentiment in Turkish social media comments.
- **🤖 Hybrid Decision Agent:** Combine SVM and LLMs for enhanced decision-making capabilities.
- **📊 Dynamic Tree Diagram Generation:** Visualize hierarchical data structures effectively.
- **📈 Comprehensive Model Performance Reports:** Get detailed metrics to evaluate machine learning models.
